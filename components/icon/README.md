# @responsive-ui/icon

> A icon component of responsive-ui.

## Install

```console
npm install @responsive-ui/icon
```

or

```console
yarn add @responsive-ui/icon
```

<br/>

## Properties

```ts
type IconProps = {
  type: string;
  style?: string;
};
```

<br/>

## Usage

```svelte
<script>
  import Icon from '@responsive-ui/icon';
</script>

<Icon type="filter" />
```
