# @responsive-ui/header

> A header component of responsive-ui.

## Install

```console
npm install @responsive-ui/header
```

or

```console
yarn add @responsive-ui/header
```

<br/>

## Properties

```ts
type HeaderProps = {
  title: string;
  style?: string;
};
```

<br/>

## Usage

```svelte
<script>
  import Header from '@responsive-ui/header';
</script>

<Header title="Component Title" />
```
