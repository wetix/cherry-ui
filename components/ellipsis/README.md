# @responsive-ui/ellipsis

> A ellipsis component of responsive-ui.

## Install

```console
npm install @responsive-ui/ellipsis
```

or

```console
yarn add @responsive-ui/ellipsis
```

<br/>

## Properties

```ts
type EllipsisProps = {
  class?: string;
  style?: string;
};
```
