# responsive-ui (alpha)

> WeTix responsive-ui component set

`responsive-ui` is a component set made by members of [WeTix](https://about.wetix.my/), and is used in our [website](https://wetix.my/) and mobile inapp.

These components are:
- Made in Svelte
- Compatible with Svelte apps

## How to install
Add this to `package.json`'s `dependencies`, or pick what you want:
```
  "@responsive-ui/accordion": "^1.1.0-alpha.17",
  "@responsive-ui/app-bar": "^1.1.0-alpha.18",
```
