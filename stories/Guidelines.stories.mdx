# Design Guideline

In `responsive-ui`, there are some pre determined breakpoints and colors that have been set. Here, we will go through them. Configuration files are planned for the future.

## Breakpoints
**Max Width:**
- sm: 576px
- md: 768px
- lg: 992px
- xl: 1200px
- sl: 1400px

## Colors

**Background Color**
- Standard: #FFF (white)
- Secondary: #f1f1f1
- Other: #FC4451

**Text Color**
- Primary: #000000 (black)
- Secondary: #505050
- Selected, Highlight, Emphasize: #FC4451
- Description: #817e7e

## Font Size

- Header: 20px
- lg: 16px
- std: 14px
- sm: 12px
- xs: 10px

## Margin / Padding

- std: 1rem
- sm: 10px
- xs: 5px
